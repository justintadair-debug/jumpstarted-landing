Setup Guide

Discord Bot Setup

Takes about 8 minutes. You only do this once.

Your progress
1

Create a Discord server

Your private server — just for you and your AI team

✓
👈
In the left sidebar of Discord, scroll all the way to the bottom. Click the + button (the big green circle with a plus).
Left sidebar → green + circle at the very bottom → "Create My Own" → "For me and my friends"
✏️
Select "Create My Own", then "For me and my friends". Name it anything — JUMPSTARTED or your own name works fine.
💡
You don't need to invite anyone. This server is just for you and your agents. It can be completely private.
2

Create a Discord application

This is where your bot lives

✓
🌐
Open a new browser tab. Go to: discord.com/developers/applications
➕
Click New Application in the top-right corner.
Top-right corner → blue "New Application" button → name it "JUMPSTARTED" → Agree → Create
📝
Name it JUMPSTARTED. Check the box agreeing to the Terms of Service. Click Create.
3

Create the bot and copy your token

The token is your bot's password — keep it private

✓
🤖
In the left sidebar of the Developer Portal, click Bot.
Left sidebar → "Bot" section
🔑
Under the TOKEN section, click Reset Token. Confirm when it asks. Then click Copy.
MTk4NjIyNDgzNDI2OTkwNTc2.GmFh3... ← yours will look like this
⚠️
Save this token right now. You can only see it once. Paste it somewhere safe before continuing. JUMPSTARTED will ask for it in the installer.
🔓
Scroll down to Privileged Gateway Intents. Turn on Message Content Intent. Click Save Changes.
Bot page → "Privileged Gateway Intents" → "Message Content Intent" toggle → Save Changes (green button)
4

Invite the bot to your server

Give the bot permission to read and send messages

✓
🔗
In the left sidebar, click OAuth2, then URL Generator.
Left sidebar → OAuth2 → URL Generator
☑️
Under Scopes, check: bot
Under Bot Permissions, check: Send Messages, Read Message History, View Channels
Scopes: bot ✓ → Bot Permissions: Send Messages ✓, Read Message History ✓, View Channels ✓
📋
Scroll to the bottom. Click Copy next to the generated URL.
🌐
Paste that URL into a new browser tab. Select your JUMPSTARTED server from the dropdown. Click Authorize. Complete the CAPTCHA.
🟢
Your bot should now appear in your server's member list (probably offline — that's fine, it goes online when JUMPSTARTED runs).
5

Turn on Developer Mode in Discord

Required to copy your channel ID in the next step

✓
⚙️
In Discord, click the gear icon ⚙️ next to your username at the bottom-left of the screen. This opens User Settings.
Bottom-left of Discord → gear icon ⚙️ next to your username → User Settings
🔧
In the left sidebar, scroll down to Advanced. Toggle Developer Mode to ON.
User Settings → "Advanced" (left sidebar, near the bottom) → "Developer Mode" toggle → ON
💡
This is the #1 step people skip. Without Developer Mode on, you cannot right-click to copy the channel ID in the next step.
6

Copy your channel ID

The ID that tells your AI team where to talk to you

✓
📢
In your JUMPSTARTED server, find the channel you want to use — typically #general. Right-click on the channel name in the left sidebar.
Left sidebar → right-click "#general" → scroll to bottom of context menu → "Copy Channel ID"
📋
At the bottom of the right-click menu, click Copy Channel ID. It will be a long number like 1234567890123456789.
1234567890123456789 ← yours will look like this
⚠️
If you don't see "Copy Channel ID" in the right-click menu, go back to Step 5 — Developer Mode is not on.

Common mistakes

These are the four things that trip people up most.

⚠️

Forgot to enable Developer Mode

Symptom: "Copy Channel ID" doesn't appear when you right-click. Fix: go back to Step 5, turn on Developer Mode, then right-click the channel again.

⚠️

Forgot to enable Message Content Intent

Symptom: bot is online but doesn't respond to messages. Fix: go to discord.com/developers/applications → your app → Bot → turn on "Message Content Intent" → Save.

⚠️

Token saved wrong (extra spaces or cut short)

Symptom: installer says "Invalid bot token." Fix: go back to the Developer Portal → Bot → Reset Token → copy again. The token is long — make sure you got the whole thing.

⚠️

Bot not added to the right server

Symptom: bot appears in the member list of the wrong server, or not at all. Fix: go to OAuth2 → URL Generator → generate a new URL → paste it in your browser and select the correct server.

✅ Test your bot

Once JUMPSTARTED is installed and running, your bot will come online and say hello. You can also test it manually:

Type anything in your Discord channel to get a response. Try:

Neo, are you there?

If Neo responds, everything is working. If nothing happens after 30 seconds, check that JUMPSTARTED is running on your Mac (run jumpstarted start in Terminal).
